# Данные меню: Холодные закуски

## Список блюд (12 позиций)

### 1. Ассорти пхали
**Презентация:** Традиционная грузинская закуска на основе овощей со специями и с грецкими орехами.
**Состав:** пхали грибной, пхали шпинатный, пхали баклажанный, пхали из болгарского перца, пхали свекольный, пури хлеб, зёрна граната, кинза, грецкий орех

### 2. Рулетики с баклажанами
**Презентация:** Запечённые баклажаны с начинкой из грецкого ореха и грузинских специй.
**Состав:** баклажаны (запечённые), паста из грецкого ореха, красная аджика, кинза, специя шераули, чеснок, винный уксус

### 3. Хумус с мацони и красной аджикой
**Презентация:** Нежная паста из нута, процеженного мацони и сладкой аджики.
**Состав:** хумус (нут), мацони, кинза, попкорн из нута, аджика сладкая, оливковое масло

### 4. Плато ассорти (холодных закусок)
**Презентация:** Традиционный набор грузинских закусок.
**Состав:** пять видов пхали, хумус, аджапсандал, соус сациви, сыр имеретинский, рулетики из баклажан, лепёшка с аджикой

### 5. Ассорти сыров
**Презентация:** Европейские и грузинские сыры.
**Состав:** пармезан, дорблю, сулугуни, имеретинский, копчёный сулугуни, мёд с кедровыми орехами, грецкий орех карамелизированный

### 6. Мясная тарелка
**Презентация:** Ассорти мясных деликатесов.
**Состав:** отварной говяжий язык, казы отварное, казы копчёное, куриный рулет с аджикой, зелёный лук, хрен

### 7. Овощная тарелка
**Презентация:** Ассорти свежих овощей.
**Состав:** огурцы, помидоры, редиска, зелёный стручковый перец, зелень

### 8. Аджапсандал
**Презентация:** Запечённые овощи с томлёными томатами. Подаётся с пури.
**Состав:** баклажан, болгарский перец, репчатый лук, томаты, перец чили, пури

### 9. Ассорти грузинских сыров
**Презентация:** Классические грузинские сыры.
**Состав:** имеретинский, сулугуни, копчёный сулугуни, микс зелени

### 10. Домашние соленья
**Презентация:** Ассорти солений.
**Состав:** капуста солёная, огурцы, малосольные помидоры, солёный чеснок, перец цицак

### 11. Сациви с курицей
**Презентация:** Куриная грудка сувид в ореховом соусе.
**Состав:** куриная грудка, грецкий орех, чеснок, кинза, аджика, куриный бульон

### 12. Вяленая тыква с муссом надуги
**Презентация:** Вяленая тыква с сыром надуги.
**Состав:** вяленая тыква, сыр надуги (мусс), соус манго, гранат, фисташки, зелёное масло, мята

## Ключевые факты для официанта

### Хиты меню (рекомендовать в первую очередь):

- **Ассорти пхали** — визитная карточка грузинской кухни
- **Плато ассорти (холодных закусок)** — идеально для знакомства с кухней
- **Сациви с курицей** — классика грузинской кухни

### Уникальные позиции:

- **Вяленая тыква с муссом надуги** — авторское блюдо, редко встречается

### Для разных запросов:

- Вегетарианцам → Пхали, Рулетики с баклажанами, Хумус, Аджапсандал
- К вину → Ассорти сыров, Мясная тарелка
- К крепким напиткам → Домашние соленья, Мясная тарелка
- Попробовать всё → Плато ассорти (холодных закусок)
